Análise de Resultados de Campanha Bancária

1. Premissas

Dado o banco de dados bank-full.csv, deseja-se realizar análise estatística, exploratória, classificatória e preditiva de forma a resolver alguns problemas de negócio.
São ao todo 6 perguntas, que foram analisadas da seguinte forma:

Perguntas de 1 a 3: Utilizei a linguagem R através do RStudio, com análises gráficas no Tableau.

Perguntas de 4 a 6: Utilizei Python através do Jupyter Notebook, com análises gráficas no matplotlib e seaborn.

Os pacotes que utilizei, tanto em R quanto em Python estão indicados em cada script.


2. Documentação

Cada pergunta possui script específico, à exceção das perguntas 2 e 3 que são complementares e estão no mesmo script.

Busquei detalhar os scripts o máximo possível com as análises/comentários para clarificar o tipo de análise que estava realizando.

Está incluído também o arquivo utilizado para as análises gráficas no Tableau e os arquivos csv gerados em cada análise.

Busquei variar as ferramentas e análises (utilizando vários métodos de classificação inclusive), para aumentar o leque de possibilidades, mas de forma alguma restrita a elas.


3. Apresentação

Para a apresentação de negócios, reuni todos os dados e explicações da forma menos técnica possível, visando uma apresentação a tomadores de decisão não-técnicos. Utilizei os resultados dos scripts, os gráficos do Tableau e Matplotlib/Seaborn e Power Point para compilação final e apresentação.

Na apresentação busquei simular a maior realidade possível, imaginando que os resultados fossem realmente ser apresentados aos tomadores de decisão de um Banco fictício ao qual chamei de XPTO, bem como o fictício time de Analytics (ao qual chamei de Lampião e Maria Bonita) ;-)  
Realizei toda a análise sozinho, sem sombra de dúvidas.
